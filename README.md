# Milhão Solidário MVP

Projeto base mobile-first inspirado em um mapa gamificado de doações.

## Como rodar

```bash
npm install
npm run dev
```

Abra no navegador:

```txt
http://localhost:3000
```

## O que já vem pronto

- Mapa Canvas com 29.000 blocos simulados
- 10 blocos centrais bloqueados
- Bloco premium de exemplo
- Bottom Sheet para Premium
- Modal central para área bloqueada
- Ranking mockado
- Admin mockado
- Checkout visual com taxa de 2%
- Prisma schema preparado para banco PostgreSQL
